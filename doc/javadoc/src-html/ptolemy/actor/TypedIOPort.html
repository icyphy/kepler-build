060/**
061 An IOPort with a type. This class implements the Typeable interface.
062 The type is represented by an instance of Type in data.type package.
063 It can be declared by calling setTypeEquals(). If this method is not
064 called, or called with a BaseType.UNKNOWN argument, the type of this port
065 will be set by type resolution using the type constraints. The type
066 constraints on this port can be specified using the methods defined in
067 the Typeable interface.
068
069 <p>This class keeps a list of TypeListeners. Whenever the type
070 changes, this class will generate an instance of TypeEvent and pass it
071 to the listeners by calling their typeChanged() method. A TypeListener
072 register its interest in the type change event of this port by calling
073 addTypeListener(), and can be removed from the listener list by calling
074 the removeTypeListener().
075
076 <p>A TypedIOPort can only link to instances of TypedIORelation. Derived
077 classes may further constrain links to a subclass of TypedIORelation.
078 To do this, they should override the protected methods _link() and
079 _linkInside() to throw an exception if their arguments are not of the
080 appropriate type.  Similarly, an TypeIOPort can only be contained by a
081 class derived from ComponentEntity and implementing the TypedActor
082 interface.  Subclasses may further constrain the containers by overriding
083 _checkContainer().
084
085 <p>Note that actors that call some of the setType<i>XXX</i> methods
086 may also need to have a clone() method.  Although the base classes
087 neatly handle most aspects of the clone operation, there are
088 subtleties involved with cloning type constraints. Absolute type
089 constraints on ports and parameters are carried automatically into the
090 clone, so clone() methods should never call setTypeEquals(). However,
091 relative type constraints of the other setType<i>XXX</I>() methods are
092 not cloned automatically because of the difficulty of ensuring that
093 the other object being referred to in a relative constraint is the
094 intended one.
095 <p> For example the Ramp actor constructor calls:
096 <pre>
097 output.setTypeAtLeast(init);
098 </pre>
099 so the clone() method of the Ramp actor calls:
100 <pre>
101 newObject.output.setTypeAtLeast(newObject.init);
102 </pre>
103

109 @Pt.AcceptedRating Yellow (neuendor)
110 */

188    /** React to a change in an attribute.  This method is called by
189     *  a contained attribute when its value changes.  This overrides
190     *  the base class so that if the attribute is an instance of
191     *  TypeAttribute, then it sets the type of the port.
192     *  @param attribute The attribute that changed.
193     *  @exception IllegalActionException If the change is not acceptable
194     *   to this container.
195     */
196    @Override
197    public void attributeChanged(Attribute attribute)
198            throws IllegalActionException {
199        if (attribute instanceof TypeAttribute) {
200            Type type = ((TypeAttribute) attribute).getType();
201
202            if (type != null) {
203                // Avoid incrementing the workspace version if the type has
204                // not changed.
205                if (!type.equals(_declaredType)
206                        || !type.equals(_resolvedType)) {
207                    setTypeEquals(type);
208                }
209            }
210        } else if (attribute == defaultValue) {
211            if (defaultValue.getToken() != null) {
212                setTypeEquals(defaultValue.getType());
213            }
214        } else {
215            super.attributeChanged(attribute);
216        }
217    }

219    /** Send a token to all connected receivers.
220     *  Tokens are in general immutable, so each receiver is given a
221     *  reference to the same token and no clones are made.
222     *  The transfer is accomplished by calling getRemoteReceivers()
223     *  to determine the number of channels with valid receivers and
224     *  then calling send() on the appropriate channels.
225     *  It would probably be faster to call put() directly on the receivers.
226     *  If there are no destination receivers, then nothing is sent.
227     *  If the port is not connected to anything, or receivers have not been
228     *  created in the remote port, then just return.
229     *  <p>
230     *  Some of this method is read-synchronized on the workspace.
231     *  Since it is possible for a thread to block while executing a put,
232     *  it is important that the thread does not hold read access on
233     *  the workspace when it is blocked. Thus this method releases
234     *  read access on the workspace before calling put.
235     *
236     *  @param token The token to send
237     *  @exception IllegalActionException If the token to be sent cannot
238     *   be converted to the type of this port
239     *  @exception NoRoomException If a send to one of the channels throws
240     *     it.
241     */
242    @Override
243    public void broadcast(Token token)
244            throws IllegalActionException, NoRoomException {
245        _checkType(token);
246        super.broadcast(token);
247    }

249    /** Send the specified portion of a token array to all receivers connected
250     *  to this port. The first <i>vectorLength</i> tokens
251     *  of the token array are sent.
252     *  <p>
253     *  Tokens are in general immutable, so each receiver
254     *  is given a reference to the same token and no clones are made.
255     *  If the port is not connected to anything, or receivers have not been
256     *  created in the remote port, or the channel index is out of
257     *  range, or the port is not an output port,
258     *  then just silently return.  This behavior makes it
259     *  easy to leave output ports unconnected when you are not interested
260     *  in the output.  The transfer is accomplished
261     *  by calling the vectorized put() method of the remote receivers.
262     *  If the port is not connected to anything, or receivers have not been
263     *  created in the remote port, then just return.
264     *  <p>
265     *  Some of this method is read-synchronized on the workspace.
266     *  Since it is possible for a thread to block while executing a put,
267     *  it is important that the thread does not hold read access on
268     *  the workspace when it is blocked. Thus this method releases
269     *  read access on the workspace before calling put.
270     *
271     *  @param tokenArray The token array to send
272     *  @param vectorLength The number of elements of the token
273     *   array to send.
274     *  @exception NoRoomException If there is no room in the receiver.
275     *  @exception IllegalActionException If the tokens to be sent cannot
276     *   be converted to the type of this port
277     */
278    @Override
279    public void broadcast(Token[] tokenArray, int vectorLength)
280            throws IllegalActionException, NoRoomException {
281        // Check types.
282        for (Token token : tokenArray) {
283            _checkType(token);
284        }
285
286        super.broadcast(tokenArray, vectorLength);
287    }

289    /** Clone this port into the specified workspace. The new port is
290     *  <i>not</i> added to the directory of that workspace (you must
291     *  do this yourself if you want it there).
292     *  The result is a new port with no connections and no container.
293     *  The new port will have the same type as this one, but will not
294     *  have any type listeners and type constraints attached to it.
295     *
296     *  @param workspace The workspace for the cloned object.
297     *  @exception CloneNotSupportedException If one or more of the
298     *   attributes cannot be cloned.
299     *  @return A new TypedIOPort.
300     */
301    @Override
302    public Object clone(Workspace workspace) throws CloneNotSupportedException {
303        TypedIOPort newObject = (TypedIOPort) super.clone(workspace);
304
305        // set _declaredType and _resolvedType
306        if (_declaredType instanceof StructuredType
307                && !_declaredType.isConstant()) {
308            newObject._declaredType = (Type) ((StructuredType) _declaredType)
309                    .clone();
310            newObject._resolvedType = newObject._declaredType;
311        }
312
313        newObject._typeTerm = null;
314        newObject._typeListeners = new LinkedList<TypeListener>();
315        newObject._constraints = new HashSet<Inequality>();
316        return newObject;
317    }

319    /** Convert the specified token into a token with type equal
320     *  to the type returned by getType(). If the token is already
321     *  of this type, then simply return the specified token.
322     *  @param token The token to convert.
323     *  @return The converted token.
324     *  @exception IllegalActionException If the conversion is
325     *   invalid.
326     */
327    @Override
328    public Token convert(Token token) throws IllegalActionException {
329        Type type = getType();
330
331        // Do not convert if automatic type conversion is disabled.
332        if (type.equals(token.getType())
333                || !this.getAutomaticTypeConversion()) {
334            return token;
335        } else {
336            try {
337                Token newToken = type.convert(token);
338                return newToken;
339            } catch (IllegalActionException ex) {
340                // Catch the exception to show the port in which it occurred.
341                throw new IllegalActionException(this, ex,
342                        "Type conversion failed.");
343            }
344        }
345    }

347    /** Return the type of this port.  If this port is opaque, this method
348     *  returns the resolved type of this port; if this port is a transparent
349     *  input port, this method returns the greatest lower bound of the types
350     *  of the inside ports; if this port is a transparent output port, this
351     *  method returns the least upper bound of the types of the inside ports.
352     *  This method is read-synchronized on the workspace.
353     *  @return An instance of Type.
354     */
355    @Override
356    public Type getType() {
357        try {
358            _workspace.getReadAccess();
359
360            Type result = BaseType.UNKNOWN;
361
362            if (isOpaque()) {
363                result = _resolvedType;
364            } else if (isInput()) {
365                // is a transparent input port. Get all the ports connected
366                // on the inside through deepInsidePortList().
367                Iterator<?> ports = deepInsidePortList().iterator();
368                Set<Type> portTypeSet = new HashSet<Type>();
369
370                while (ports.hasNext()) {
371                    TypedIOPort port = (TypedIOPort) ports.next();
372
373                    // Rule out case where this port itself is listed...
374                    if (port != this && port.isInput()) {
375                        portTypeSet.add(port.getType());
376                    }
377                }
378
379                CPO lattice = TypeLattice.lattice();
380                result = (Type) lattice.greatestLowerBound(portTypeSet);
381            } else if (isOutput()) {
382                // is a transparent output port. Get all the ports connected
383                // on the inside through deepInsidePortList().
384                Iterator<?> ports = deepInsidePortList().iterator();
385                List<Type> portTypeList = new LinkedList<Type>();
386
387                while (ports.hasNext()) {
388                    TypedIOPort port = (TypedIOPort) ports.next();
389
390                    // Rule out case where this port itself is listed...
391                    if (port != this && port.isOutput()) {
392                        portTypeList.add(port.getType());
393                    }
394                }
395
396                CPO lattice = TypeLattice.lattice();
397                result = (Type) lattice
398                        .leastUpperBound(new HashSet<Type>(portTypeList));
399            }
400
401            return result;
402        } finally {
403            _workspace.doneReading();
404        }
405    }

423    /** Check whether the current type of this port is acceptable.
424     *  A type is acceptable if it represents an instantiable object.
425     *  Any type is acceptable (including non-instantiable types)
426     *  if the associated port is not connected to anything.
427     *  @return True if the current type is acceptable.
428     */
429    @Override
430    public boolean isTypeAcceptable() {
431        if (this.getType().isInstantiable()) {
432            return true;
433        }
434
435        // For a disconnected port, any type is acceptable.
436        if (this.numLinks() == 0) {
437            return true;
438        }
439
440        // For an input port with no sources, any type is acceptable.
441        if (isInput() && !isOutput() && numberOfSources() == 0) {
442            return true;
443        }
444
445        return false;
446    }

467    /** Send a token to the specified channel, checking the type
468     *  and converting the token if necessary.
469     *  If the port is not connected to anything, or receivers have not been
470     *  created in the remote port, or the channel index is out of
471     *  range, or the port is not an output port,
472     *  then just silently return.  This behavior makes it
473     *  easy to leave output ports unconnected when you are not interested
474     *  in the output.
475     *  If the type of the specified token is the type of this
476     *  port, or the token can be converted to that type
477     *  losslessly, the token is sent to all receivers connected to the
478     *  specified channel. Otherwise, IllegalActionException is thrown.
479     *  Before putting the token into the destination receivers, this
480     *  method also checks the type of the remote input port,
481     *  and converts the token if necessary.
482     *  The conversion is done by calling the
483     *  convert() method of the type of the remote input port.
484     *  <p>
485     *  If the token argument is null, then no token is sent.
486     *  What this means exactly is domain dependent. In some domains
487     *  (SR, Continuous), it asserts that the output is "absent."
488     *  That is, {@link #isKnown(int)} will return true
489     *  and {@link #hasToken(int)} will return false.
490     *  In other domains (SDF, DE, PN), it simply does nothing.
491     *  Nothing is sent. Hence, {@link #isKnown(int)}
492     *  and {@link #hasToken(int)} will return whatever they would
493     *  have returned before the call.
494     *  <p>
495     *  Some of this method is read-synchronized on the workspace.
496     *  Since it is possible for a thread to block while executing a put,
497     *  it is important that the thread does not hold read access on
498     *  the workspace when it is blocked. Thus this method releases
499     *  read access on the workspace before calling put.
500     *
501     *  @param channelIndex The index of the channel, from 0 to width-1.
502     *  @param token The token to send, or null to send no token.
503     *  @exception IllegalActionException If the token to be sent cannot
504     *   be converted to the type of this port, or if the token is null.
505     *  @exception NoRoomException If there is no room in the receiver.
506     */

508    public void send(int channelIndex, Token token)
509            throws IllegalActionException, NoRoomException {
510        if (token != null) {
511            _checkType(token);
512        }
513        super.send(channelIndex, token);
514    }

549     *  @param vectorLength The number of elements of the token

622    /** Allow actors to disable automatic type conversion on their input
623     *  ports in case they do not need it. For example, AddSubtract and
624     *  Display accept any token type, because they make use of methods

678     *  of Type. If the type is BaseType.UNKNOWN, the determination of the type
679     *  is left to type resolution.

682     *  clone() method.  Absolute type constraints on ports and
683     *  parameters are carried automatically into the clone, so
684     *  clone() methods of actors should never call setTypeEquals().
685     *  Actors that call the other setType<i>XXX</i>() methods should
686     *  have a clone() method.
687     *
688     *  <p> This method is write-synchronized on the workspace.
689     *  @param type A Type.
690     */

692    public void setTypeEquals(Type type) {
693        try {
694            _workspace.getWriteAccess();
695
696            try {
697                _declaredType = (Type) type.clone();
698            } catch (CloneNotSupportedException cloneNotSupported) {
699                throw new InternalErrorException(this, cloneNotSupported,
700                        "TypedIOPort.setTypeEquals: Cannot clone type");

703            // Note: we are careful here to set the type before notifying
704            // type listeners.

708            // in initialize() where we handled StructuredTypes?

852    /** Check that the specified token as well as the token in
853     *  the default value, if specified, is compatible with the
854     *  resolved type of this port. If the resolved type is UNKNOWN
855     *  then we have to assume unknown is acceptable (e.g. the port
856     *  is not connected to anything), so we accept any token type.
857     *  If the type is GENERAL, then we don't check the type because
858     *  all types are lower and none are incomparable to GENERAL.
859     *  @param token The token to check.
860     *  @exception IllegalActionException If the specified token is
861     *   either incomparable to the resolved type or higher in the
862     *   type lattice.
863     */
864    protected void _checkType(Token token) throws IllegalActionException {
865        // CapeCode: Check for GENERAL here and avoid a possible hang when
866        // calling localFunctions.js convertToToken() and we fall through
867        // to a possibly very large RecordToken.
868        if (_resolvedType.equals(BaseType.UNKNOWN)
869                || _resolvedType.equals(BaseType.GENERAL)) {
870            return;
871        }
872        int compare = TypeLattice.compare(token.getType(), _resolvedType);
873
874        if (compare == CPO.HIGHER || compare == CPO.INCOMPARABLE) {
875            throw new RunTimeTypeCheckException(this, token);
876        }
877
878        if (defaultValue.getToken() != null) {
879            compare = TypeLattice.compare(defaultValue.getToken().getType(),
880                    _resolvedType);
881
882            if (compare == CPO.HIGHER || compare == CPO.INCOMPARABLE) {
883                throw new RunTimeTypeCheckException(this,
884                        defaultValue.getToken());
885            }
886        }
887    }

889    /** Return a description of the object.  The level of detail depends
890     *  on the argument, which is an or-ing of the static final constants
891     *  defined in the NamedObj class and in this class.
892     *  Lines are indented according to
893     *  to the level argument using the protected method _getIndentPrefix().
894     *  Zero, one or two brackets can be specified to surround the returned
895     *  description.  If one is specified it is the leading bracket.
896     *  This is used by derived classes that will append to the description.
897     *  Those derived classes are responsible for the closing bracket.
898     *  An argument other than 0, 1, or 2 is taken to be equivalent to 0.
899     *  <p>
900     *  If the detail argument sets the bit defined by the constant
901     *  TYPE, then append to the description a field of the form
902     *  "type {declared <i>declaredType</i> resolved <i>resolvedType</i>}".
903     *  The declared type is the type set through setTypeEquals(). If this
904     *  method is not called, the declared type is BaseType.UNKNOWN.
905     *  The resolved type is the type of this port.  Both types are
906     *  represented by the names of the corresponding tokens.
907     *  <p>
908     *
909     *  This method is read-synchronized on the workspace.
910     *  @param detail The level of detail.
911     *  @param indent The amount of indenting.
912     *  @param bracket The number of surrounding brackets (0, 1, or 2).
913     *  @return A description of the object.
914     *  @exception IllegalActionException If thrown while getting the
915     *  description of subcomponents.
916     */
917    @Override
918    protected String _description(int detail, int indent, int bracket)
919            throws IllegalActionException {
920        try {
921            _workspace.getReadAccess();
922
923            String result;
924
925            if (bracket == 1 || bracket == 2) {
926                result = super._description(detail, indent, 1);
927            } else {
928                result = super._description(detail, indent, 0);
929            }
930
931            if ((detail & TYPE) != 0) {
932                if (result.trim().length() > 0) {
933                    result += " ";
934                }
935
936                result += "type {declared ";
937                result += _declaredType.toString();
938                result += " resolved ";
939                result += getType().toString();
940                result += "}";
941            }
942
943            if (bracket == 2) {
944                result += "}";
945            }
946
947            return result;
948        } finally {
949            _workspace.doneReading();
950        }
951    }

971    private void _notifyTypeListener(Type oldType, Type newType) {

973            TypeEvent event = new TypeEvent(this, oldType, newType);

1143         *  @exception IllegalActionException If the new type violates
1144         *   the declared type of this port.
1145         */
1146        @Override
1147        public void setValue(Object type) throws IllegalActionException {
1148            // Cannot set value if declared type is a constant
1149            if (!isSettable()) {
1150                throw new IllegalActionException(
1151                        "TypedIOPort$TypeTerm.setValue: The type is not "
1152                                + "settable.");
1153            }
1154            // The argument type must be a substitution instance for the
1155            // declared type, this means:
1156            // - for basic types: the argument is equal to declared type,
1157            // or declared type is unknown
1158            // - for structured types: if the shape of the structure of
1159            // argument and declared type are the same and leafs of the
1160            // structure of the argument type, which must be basic
1161            // types, all are substitution instances for the corresponding
1162            // element types found in the declared type
1163            // So the declared type must be unknown, equal to, or identically
1164            // structured as the argument type (where elements in the declared
1165            // type are allowed to be left unknown).
1166            if (!_declaredType.isSubstitutionInstance((Type) type)) {
1167                throw new IllegalActionException("Type conflict on port "
1168                        + TypedIOPort.this.getFullName() + ".\n"
1169                        + "Declared type is " + _declaredType.toString() + ".\n"
1170                        + "The connection or type constraints, however, "
1171                        + "require type " + type.toString());
1172            }
1173
1174            Type oldType = _resolvedType;
1175
1176            // The type is settable, so declared type must be unknown, or
1177            // a structured type with one or more unknown elements.
1178            if (_declaredType == BaseType.UNKNOWN) {
1179                _resolvedType = (Type) type;
1180            } else {
1181                // _declaredType is a StructuredType
1182                ((StructuredType) _resolvedType)
1183                        .updateType((StructuredType) type);
1184            }
1185
1186            if (!oldType.equals(type)) {
1187                _notifyTypeListener(oldType, _resolvedType);
1188            }
1189        }
